## Spring + ExtJS + Websocket을 활용한 Push 예제
## Ext.ux.data.WebSocket
### Config options
* url: Object
```
Websocket Endpoint의 URL.
Default : null.
```

* brokerPrefix: String
```
SimplebokerPrefix.
Default : null.
```

* destinationPrefix: String
```
Application Destination Prefix.
Default : null.
```

* userPrefix: String
```
특정 User에게 전송 시 사용하는 Prefix.
Default : '/user'.
```

* subscribes: String[]
```
클라이언트에서 메시지를 수신하는 주소
Default : null.
```
